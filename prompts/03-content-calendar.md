# Prompt 3 — 90-Day Content Calendar + Issue Template System

**Never stare at a blank page again. Never run out of ideas. Never publish something that misses the point.**

---

## The Prompt

```
You are a content strategist who runs a newsletter with 50,000 subscribers. You know exactly how to build a content system that never runs dry, keeps readers opening every issue, and builds toward a monetization goal.

MY NEWSLETTER:
- Name: [NEWSLETTER NAME]
- Niche statement: [PASTE FROM PROMPT 1]
- Publishing cadence: [weekly / biweekly / 3x/week]
- My monetization goal (90 days): [e.g., 500 paid subscribers / 2 sponsors / sell a course]
- Content formats I'm comfortable with: [essays / lists / case studies / interviews / data breakdowns / tutorials]
- Topics I never want to cover: [optional — your off-limits list]

DELIVER:

1. 52-WEEK CONTENT THEME SYSTEM
Organize the year into 4 quarters × 13 weeks each. For each quarter:
- Quarter theme (the macro story arc)
- 3 core topic pillars for that quarter
- 1 "signature piece" concept (your biggest, most shareable issue of the quarter)

2. 90-DAY DETAILED CALENDAR
Issues 1 through 26 (assuming biweekly) or 1 through 13 (weekly). For each issue:
- Week number + publish date template (e.g., "Week 3 — Issue #5")
- Working title (compelling, specific, not clickbait)
- Content format (essay / list / case study / interview / data / tutorial / hybrid)
- 1-sentence hook (why a reader opens this NOW)
- 3 key points to cover
- Monetization micro-goal for this issue (build list / drive upgrade / promote product)

3. EVERGREEN SERIES STRUCTURE
3 recurring series that run every 4-6 weeks — fills calendar gaps, trains reader expectations, builds brand.
For each series:
- Series name + tagline
- Format description (what it always includes)
- First 3 issue titles
- How it drives revenue (list growth / paid conversion / sponsor mention)

4. MASTER ISSUE TEMPLATE
A 3-section framework for every issue:
SECTION 1 — HOOK (200-400 words): Opening hook formula, 3 hook types that work for my niche, examples
SECTION 2 — BODY (600-1,200 words): Structure options by format type, transition sentences, "so what" integration
SECTION 3 — CTA (50-150 words): 3 CTA templates by goal (subscribe / upgrade / share / buy)

5. RE-ENGAGEMENT SEQUENCE
A 3-email sequence for subscribers who haven't opened in 60+ days:
- Email 1: "Did we lose you?" (Day 60) — subject line + body
- Email 2: "Here's what you missed" (Day 67) — curated best-of
- Email 3: "Last chance" (Day 74) — sunset or stay
```

---

## How to Use This Output

- Drop the 52-week theme system into a Notion or Airtable database as your editorial master plan
- Schedule all 90 days in your publishing calendar before you write Issue #1
- Use the evergreen series to fill "low energy" weeks without sacrificing quality
- Apply the issue template to every draft — it takes 20 minutes off writing time per issue
- Set up the re-engagement sequence as an automation trigger at 60 days of no-open

---

## Pro Tips

- **Title the issues specifically.** "5 Things About Leadership" fails. "Why the Worst IT Managers All Make the Same Mistake in Month 3" works.
- **One CTA per issue.** Multiple CTAs dilute action. Pick the most important one for that week.
- **The re-engagement sequence is revenue.** Lapsed subscribers who re-engage convert to paid at 2-3× the rate of new subscribers.
- **Your signature quarterly piece is a launch asset.** Promote it on social, guest post excerpts, use it as a lead magnet.
